# Getting Started
A simple blackjack game made in python
### Prerequisites
* python
  ```sh
  python
  ```

### Installation

1. Clone the repo
   ```sh
   git clone https://github.com/mKupiec2/blackjack-py.git
   ```
2. Run the game
   ```sh
   python blackjack.py
   ```
